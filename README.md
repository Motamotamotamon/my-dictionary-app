A personal English dictionary app built with Next.js and Prisma.  
You can add, search, edit, and delete your own vocabulary.

## Features

- Add new words with meaning, part of speech, definition, and example
- Search by word or meaning
- Edit meaning and example inline
- Delete words
- Persistent storage with database (Prisma)

## Tech Stack

- Next.js (App Router)
- React
- Prisma
- SQLite (or your current DB)
